# 测试标题

## Introduction

这是一个测试文件，用于测试翻译。

## Usage

运行 `python main.py` 来执行程序。

## Functionality

该程序将读取 `input.txt` 文件并将其转换为大写，然后将结果写入 `output.txt` 文件。

## Example

例如，如果 `input.txt` 包含以下内容：

```
hello world
```

然后运行程序，`output.txt` 将包含以下内容：

```
HELLO WORLD
```

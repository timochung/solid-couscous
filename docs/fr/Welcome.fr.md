## Getting Started

To use the library, you must first install it.

Pour utiliser la bibliothèque, vous devez d'abord l'installer.

```bash
pip install my-library
```

Once installed, you can import it into your Python code:

Une fois installée, vous pouvez l'importer dans votre code Python :

```python
import my_library
```
